# Task Manager Client

Angular SPA for the Task Manager application.

## Run

```powershell
npm install
npm start
```

App: http://localhost:4200

## API Configuration

Update `src/environments/environment.ts` if the backend URL changes:

```typescript
apiUrl: 'https://localhost:7156/api'
```

## Build

```powershell
npm run build
```

Output: `dist/task-manager-ui/`
